# README
## 主要功能
本项目主要用于实时数据更新情况的展示及提醒功能，目前主要包括：
- 实时数据更新表格显示
- 近七天数据更新趋势图
- 本地服务器占用情况图
- 本地服务器数据占用超过80%实时邮件提醒
- 每周邮件发送数据更新报告
